# Calendar Widget

A lightweight calendar widget that lives in the system notification area, providing quick access to a calendar view with emphasis on holiday visibility and vacation planning.

## Features

- System tray icon for quick access
- Calendar view with holiday highlighting
- Vacation planning (placeholder)

## Installation

1. Install Java 11+
2. Install Maven
3. Run: `mvn clean compile`

## Usage

Run: `mvn exec:java`

Right-click the tray icon to show calendar.
